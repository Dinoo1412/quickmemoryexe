## 协同之道，如何在本地运行？
1. fork我的仓库
2. git clone自己的仓库，放在本地
3. 下载nodejs和npm（如果有可以跳过）
4. npm install下载依赖
5. node server.js 打开后端文件
6. npm start 运行！
